# AquaFaste

**Hydration tracker iOS app — sister app to Lumifaste (fasting tracker).**

## What It Is

AquaFaste is a clean, ad-free hydration tracking app for iOS. Users log water intake, track daily goals, and build healthy drinking habits. Part of the "Faste" app family alongside Lumifaste.

## Current State

Research phase — M001 in progress.

## Stack

- SwiftUI + SwiftData
- StoreKit 2 (native IAP)
- HealthKit (dietaryWater)
- iOS 17+
- No third-party dependencies

## Core Principles

- **No ads ever** — premium subscription model only
- **Generous free tier** — core tracking free, premium for power users
- **Cross-promote with Lumifaste** — same brand family, same developer
- **Privacy first** — no tracking, no analytics beyond basic telemetry
